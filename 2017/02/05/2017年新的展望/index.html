2017年新的展望

发表于 2017-02-05 | 分类于
随笔

掐指一算，毕业也已经三年了。从2013年懵懵懂懂得从大学毕业，到现在慢慢觉得进入正轨。

学习

2016年在工作之余接了个项目做，本来想能够利用业余时间赚点外快旅游钱，结果项目从一开始就需求不明确，做到一半的时候反而出现需求晚于开发的情况。然后等到我们交出几个版本之后，却反而需要我们推倒以前的逻辑，导致项目耗时远远大于原来的预计时间。

另外，程序员是需要不断充电的群体，尤其是前端这个不断发展的行业。业余时间被占用后，导致没有时间来进行自我充电。新的一年，希望自己能够利用好时间提升自己。2017年的学习重点依然是Vue和React等前端框架，希望自己做好一个自己的web app.至少熟练掌握一个技术栈，现在主要学习Vue2 Vuex Vue-router的Vue栈。其次是nodejs的学习，希望能够系统性的学习一下，以前只用过express，sails等框架。然后继续做自己的python爬虫，希望能够爬点更加有用有趣的数据出来。

既然有了这个博客的平台，希望自己能够认真维护下去，不管是原创的还是转帖的文章，希望自己能够多发博文。

阅读

阅读渐渐得变成生活中不可或缺的一个主题了，2016年除了技术书籍之外，印象最深的是那边红极一时的《人类简史》和吴晓波的《激荡三十》，当然还有看了很久的《三体》和看了电影才去看的千禧年三部曲《龙纹身的女孩》。

其中《人类简史》根本不像是名字听起来一样一本枯燥琐碎的科普书，反正涉及丰富的考古学、生物学和历史学等专业知识，没有专业的晦涩而是通俗易懂的论证作者的理论。全书看下来，感觉自己作为一个人类一路跋涉了十万年一样。

不敢给自己定明年阅读多少本书的目标，因为总会给自己找完不成的理由，但是买书看书以及是一个不能割舍的习惯了。

上海

工作三年来第一次有了这样的想法。不否认从13年刚到上海到现在的2017年，自己已经慢慢变得不像以前的自己了。看着从2016年开始，上海带领着全国大小城市，房价蹭蹭不断攀登者高峰，内心第一次觉得阶级的鸿沟是那么得难以跨越。身边的同事也一个一个赶紧买房或者考虑着离开上海，我自己也慢慢变得不那么淡定。以前一直觉得自己年纪还小，可一转眼也已经26了，进入了二十岁的下半页。买房安定下来渐渐变成一件不是那么可有可无的事情了。

去年在老家买了房，现在每个月缴纳着不算多的房贷，这是一种复杂的幸福感和紧迫感，买房的时候自己看了无数楼盘，问了很多中介，感觉到自己已经从一个学生思维的年轻人变成一个成熟的社会人的改变，这种改变可能慢慢会驱使我自己做出更加合理的选择吧。

拓宽自己的社交

作为程序猿，总给人是宅男的印象。其实大多数的时候，一个人写着代码兴奋但是孤独的。其实我们总是用社交来掩饰内心的孤独和不自信吧，但是自己也是那种闲不下来的性格，希望2017年能够找到自己喜欢的圈子认识更多有趣的人，能够在社交中找到内心的平静。
