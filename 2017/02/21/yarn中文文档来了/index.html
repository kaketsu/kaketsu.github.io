发表于 2017-02-21 | 分类于

Yarn

Yarn 是Facebook推出的一款开源JavaScript包管理器。

Yarn 是用来代替 npm 客户端的命令行工具，它更快，更可靠，更安全。用 Yarn 后可以继续使用你所有的包。

去年看到 Yarn 之后，稍微用了一下也没有注意，今天看到 Yarn 终于推出中文文档了。

详细信息可以看这个 Yarn 的中文网站。

目前项目用 Yarn 基本可以无痛迁移使用，所以如果受够了 npm 的速度，可以尝试使用下 Yarn 。
